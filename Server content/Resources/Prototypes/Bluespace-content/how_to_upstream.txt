1. Вам необходимо проверить ремоуты через git remote -v
2. Вам необходимо отрезать ремоут через git remote rm upstream
3. Вам необходимо добавить проект 2 через git remote add proj2 https://github.com/space-syndicate/space-station-14.git
4. Вам необходимо замержить ветку git merge proj/master
5. Далее решаете конфликты
6. Потом удаляете proj через git remote rm proj2
7. Затем добавляете обычную ветку, мержите ее, решаете конфликты
8. Затем пушите изменения на удаленный репозиторий
9. Создаете ПР
10. Profit!
